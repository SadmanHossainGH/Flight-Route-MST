1. Assignment Description

1. In this assignment, you will implement the Kruskal's algorithm to find a minimum spanning tree and compute its total weight. The example input is given as follows:

7    9
Atlanta,Boston(700.5),Chicago(250.5),
Boston,Dallas(350.0),El Paso(400.75),
Chicago,Houston(625.0),
Dallas,Houston(600.5),Kansas(450.75),
El Paso,Kansas(300.75),
Houston,Kansas(550.0),
Kansas,,
End

The two numbers in the first line represents number of cities and number of flight routes. i.e. they corresponds to number of vertices and number of edges of an undirected graph. For above example, there will be 7 different cities and 9 different round flight routes.

Followed by the two numbers are lines repersent departure and arrival city and air ticket price info. For example:

Atlanta,Boston(700.5),Chicago(250.5),

Above represents a departure city Atlanta and two arrival cities, Boston and Chicago. Number inside the parenthesis represents the airfair, i.e. from Atlanta to Boston, the air ticket price is $700.5, and from Atlanta to Chicago, the air ticket price is $250.5. Note: the ticket price info. represents the weight of the edge.

Each edge of the graph will be the round route from a departure city to an arrival city. In above input file, we only list each route once. For example there's one route from Atlanta to Boston and ticket price is 700.5, we don't list route from Boston back to Atlanta since Kruskal algorithm only handle undirected graph.

Each input will end with an "End".

2. Kruskal's algorithm uses a data structure called disjoint-set. A disjoint-set data structure is a data structure that keeps track of a set of elements partitioned into a number of disjoint (non-overlapping) subsets. In this assignment, we will use disjoint-set forest to implement the Kruskal's algorithm. Read our textbook chapter 21.3 (pp.568) for more information on Disjoint-set forests.

For a faster implementation of the disjoint sets, you will need to design the following operations:

  • make_set(x) : creates a new set whose only member (and thus representative is x). Since the sets are disjoint, we require that x not already be in some other sets.

  • find_set(x): returns a pointer to the repersentative of the (unique) set containing x.

  • Union(x): unites the dynamic sets that contain x and y, say Sx and Sy, into a new set that is the union of these two sets. We assue the two sets are disjoint prior to the operation.

3. Using the given input, store the information of edges inside an edge array. For each edge, it contains the two cities' info. and also the weight of the edge (price). 

4. In Kruskal's algorithm, you will need to sort all edge in non-decreasing order according to their weight. In this assignment, we will use Merge Sort to do this, feel free to switch to Heap Sort if you'd like. Then use the disjoint-set data structure to produce a minimum spanning tree using Kruskal's algorithm and print the minimum spanning tree edges as follows:

MST Kruskal Algorithm Result

Atlanta    <--->    Chicago    = 250.50
El Paso    <--->    Kansas     = 300.75
Boston     <--->    Dallas     = 350.00
Boston     <--->    El Paso    = 400.75
Houston    <--->    Kansas     = 550.00
Chicago    <--->    Houston    = 625.00
============================================
Total Price: 2477.00

print all vertex pairs (edges) with their weight in the order that they were added to the tree, and the total tree weight. Note: in the Kruskal's algorithm, when an edge is added to the set A:

A = A U { ( u, v ) }

you just need to print each edge (with their vertex names and its weight) instead of actually adding to the set A, so that your Kruskal function does not need to return the set A.
